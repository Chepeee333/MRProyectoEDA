Este prototipo / sistema es desarrollado para la pequeña empresa de La Eskina Del Antojo, se trata de un sistema de 
punto de venta donde se pueden hacer altas, bajas y cambios de distintas tablas de una base de datos, se maneja
facilmente e intuitivamente desde la apariencia sencilla y bonita de su menu.

-----GUIA DE USO----

--PANTALLA DE PUNTO DE VENTA--

Aqui se pueden agregar productos a un pedido solo dandole click arriba, ademas estan
organizados por categorias para que sea mas facil encontrar los productos. Una vez
agregados se escribe el nombre de la persona asociada a ese pedido y se ingresa con
cuanto dinero pago el cliente y eso te calculara el cambio que debes de dar, despues
ese pedido se guarda para otros registros

--PANTALLA DE PRODUCTOS--

Aqui se pueden ver todos los productos que estan en el sistema, se pueden editar, borrar
y consultar, ademas arriba hay un boton para agregar mas productos donde se tienen que ingresar
todos los datos del producto que se desea ingresar al sistema.

--PANTALLA DE CLIENTES--

Aqui se pueden ver todos los clientes que estan en el sistema, se pueden editar, borrar
y consultar, ademas arriba hay un boton para agregar mas clientes donde se tienen que ingresar
todos los datos del cliente que se desea ingresar al sistema.

--PANTALLA DE GANANCIAS--

Aqui se pueden ver las estadisticas del dia como las ganancias, el promedio de ganancia de los
pedidos y el total de pedidos de ese dia ademas de ver los pedidos que se han hecho de todos los tiempos
ademas de ver una grafica con las ventas de cada dia de la semana presente.
